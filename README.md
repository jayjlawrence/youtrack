# Youtrack::Ruby
Youtrack API client


## Installation
Add this line to your application's Gemfile:

```ruby
gem 'youtrack', github: 'maxivak/youtrack'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install youtrack-ruby
```
